# Antares_Uniandes

The "antares_uniandes.dat" file summerize the radial velocity measurements from a total of 580 spectra of Antares over 153 nights, taken from the Universidad de los Andes between 2015 and 2020 using the ESPARTACO spectrograph.

The analysis of this dataset and technical description of its adquisition is contained in the paper: "Preliminary results from 5 years' spectral monitoring of Antares" submitted to the Revista Mexicana de Astronom√≠a by Oostra, B. & Batista M. 2024.
